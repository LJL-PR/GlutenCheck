Architecture Diagram for Gluten-Free Checker

graph LR A[User] -->|HTTP Request| B[Flask Application] B -->|Serves HTML| C[Templates Folder] B -->|Handles Data| D[SQLite Database] B -->|Loads Dependencies| E[Static Folder] C -->|HTML Templates| F["index.html"] C -->|HTML Templates| G["search_results.html"] E -->|CSS/JS Files| H[CSS and JS] D -->|Stores Data| I["food_data.db"] D -->|Populated by Script| J["populate_from_csv.py"]

The architecture consists of:

  • Flask Application: Handles HTTP requests and routes.
  • Templates: Contains HTML files like index.html and search_results.html.
  • SQLite Database: Stores food data in food_data.db.
  • Static Files: Includes CSS and JS for styling and interactivity.
  • CSV Population Script: populate_from_csv.py populates the database from gluten_data.csv.
